HomeMatic WhatsApp Addon for the Raspimatic

This repository hosts the development on a HomeMatic Raspimatic-Addon to send WhatsApp-Messages by using the Yowsup python library.

Supported CCU Devices

An installation of the Addon is only supported on the following CCU devices:

  • RaspberryMatic (beta)
Installation
  1. Download the latest Addon release on github
  2. Upload it via the HomeMatic WebUI provided Addon functionality.
  3. Login to your CCU and configure the Addon accordingly.
Support

If you encounter any problems or have any idea for enhancements of this addon please create a corresponding ticket at the issue tracker. For getting direct help a german speaking discussion forum is hosting a seperate discussion thread as well here.
