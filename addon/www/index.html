FRITZ!-basierte Anwesenheitserkennung für HomeMatic

Dieses Repository beinhaltet die Entwicklung eines HomeMatic Raspimatic-Addon, mit dem man WhatsApp-Nachrichten senden kann, unter Zuhilfename der Yowsup Python Bibliothek.

Unterstützte CCU Platformen

Eine Installation des Addons ist nur ünterstützt für die:

  • RaspberryMatic (beta)
Installation
  1. Download das letzte Release auf Github
  2. Hochladen des Addons auf die Raspimatic.
  3. Konfigurieren des PlugIns auf der Raspimatic.
Support

Bei Problemen oder Ideen zur Weiterentwicklung erstelle ein Ticket auf issue tracker. Für Hilfe gibt es ein deutschsprachiges Forum auf Homematic.

Lizenz

Die Lizenz des Plugins unterliegt der GPL license.
